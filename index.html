📖 Instruções de Uso
Guia completo do sistema

1. Como funciona

🌐Este sistema roda 100% no navegador — sem servidor, sem Python, sem instalar nada. Basta abrir o arquivo HTML.
💾Os dados ficam salvos no IndexedDB — banco de dados nativo do navegador. Os dados persistem entre sessões automaticamente.
⚠️Limpar o cache/dados do navegador apaga o banco. Use o botão "Exportar Cadastro" para fazer backup periódico.

2. Fluxo de uso

1️⃣Cadastro: verifique os funcionários em Cadastro Geral e nas abas de turno. Adicione, edite ou remova diretamente na tabela.
2️⃣Importar TXT: acesse Importar TXT (AFD), arraste os arquivos .txt e clique em Importar. O sistema lê, valida e salva no IndexedDB.
3️⃣Relatório por Turno: clique em Gerar — baixa um XLSX com Relat T1/T2/T3 (detalhado) e Resumo T1/T2/T3 (consolidado).
4️⃣Espelho por Turno: selecione os turnos e gere o espelho individual com até 8 batidas por dia, total trabalhado e campo de expediente.

3. Formato AFD (Portaria 1510)

ℹ️ Somente linhas com tipo 3 (posição 10) são processadas — marcações de ponto. Outros tipos são ignorados.
PosiçãoCampoStatus
1–9NSR — Nº Sequencial de Registro✅ Lido
10Tipo ("3" = batida de ponto)✅ Filtrado
11–12Dia (DD) 01–31✅ Validado
13–14Mês (MM) 01–12✅ Validado
15–18Ano (AAAA) 2000–2099✅ Validado
19–20Hora (HH) 00–23✅ Validado
21–22Minuto (MM) 00–59✅ Validado
24–34PIS/NIS (11 dígitos)✅ Chave de cadastro

4. Espelho — legenda de cores

📅Col A/B = Data e dia da semana (SEG…DOM)
⏱️Cols C–J = Ponto 1 a Ponto 8 (até 8 batidas/dia)
🟢Col K = Total trabalhado: (P2-P1)+(P4-P3)+...
✏️Col L = Expediente — preencher manualmente em h:mm
📊Col M = Saldo = K − L
🟠Linha laranja = batida 00:xx → exibida como 24:xx (turno noturno)
🔴Total negativo → corrigir antes de validar

5. Banco de dados (IndexedDB)

🗃️Store cadastro: funcionários com PIS, matrícula, nome, cargo, turno
🗃️Store registros: batidas importadas dos arquivos AFD
🗃️Store log: erros de validação durante importação
🔒Deduplicação por chave: PIS|Data|Hora|NSR — nunca duplica
⬇️Botão "Exportar XLSX" no cadastro faz backup das 4 abas
🔄Botão "Resetar DB" apaga tudo e reimporta o cadastro original de 126 funcionários

6. GitHub / Distribuição

📁Este arquivo é autocontido — basta distribuir o .html
🌐Pode hospedar no GitHub Pages: basta colocar o HTML em um repositório e ativar Pages
